# 安装说明

这是一个可分享的 Claude / Codex Skill 包。

## 安装到 Claude

1. 解压 `gongzhonghao-typeset-skill.zip`。
2. 把解压出来的 `gongzhonghao-typeset` 文件夹放到：

```text
~/.claude/skills/gongzhonghao-typeset
```

Windows 示例：

```text
C:\Users\你的用户名\.claude\skills\gongzhonghao-typeset
```

3. 重启 Claude / Codex，或重新打开会话。

## 使用

默认品牌名：佳敏在折腾

默认主题：温暖棕

更多用法见：

```text
USER_GUIDE.md
```

## 三套主题

- 温暖棕：`themes\brown.json`
- 沉稳绿：`themes\editorial.json`
- 清新蓝：`themes\fresh.json`
